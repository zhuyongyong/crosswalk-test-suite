Precondition:

  1. Require the latest Cordova CLI, and must >= 5.0.0
    $ sudo npm install cordova -g
  2. Install BeautifulSoup
    $ sudo pip install BeautifulSoup

Test Steps:

  1. Enter 'cordova-lite-android-tests/res' directory
  2. Run the python script:(take 'arm' as example, if your device is x86, need to use 'x86' instead of 'arm')
    $ python test.py -a arm
  3. Check the command output
  4. Check if there are several apk files in 'cordova-lite-android-tests/res directory
    There should be 14 apk files, and the apk names start with 'CrosswalkVersion_canary'
  5. Install and launch these apks one by one
  6. Check if 'Expected Crosswalk Version' is the same with 'Actual Crosswalk Version' on each app homepage

Expected Output:

  1. The command run successfully, no error
  2. There are several apk files in the directory
  3. Every apk can be installed successfully
  4. Every apk can launch successfully
  5. Every apk can run successfully
  6. "Expected Crosswalk Version" is the same with "Actual Crosswalk Version" on each app homepage

Test Steps:

  1. Get sampleRelease.apk from cordova_sampleapp.zip(you can download it from internal release link)
  2. Generate a private key using keytool:
    3. $ keytool -genkey -v -keystore orgtestfoo-release-key.keystore -alias mytestfoo -keyalg RSA -keysize 2048 -validity 10000
  3. Sign app with the private key using jarsigner:
    4. $ jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore orgtestfoo-release-key.keystore sampleRelease.apk mytestfoo
  4. Verifies if the APK is signed:
    5. $ jarsigner -verify -verbose -certs sampleRelease.apk
  5. Align the final APK package using zipalign:
    6. $ zipalign -v 4 sampleRelease.apk sampleRelease-signed.apk
  6. Install the signed apk
  7. Launch the app
  8. Exit the app
